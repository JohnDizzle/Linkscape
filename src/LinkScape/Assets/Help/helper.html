Meet Linker

Linker is the browser helper inside LinkScape. It can answer questions about local browser data, use LinkScape tools, and optionally hand general chat questions to a provider key you choose.

Local tools first then provider chat when connected

Local MCP tools

Read browser history, favorites, saved tabs, active tabs, collections, and browser settings stored by LinkScape.

Browser actions

Switch tabs, open a new tab, navigate, go home, reload, and answer from the current LinkScape window.

Optional AI services

Add your own provider key to ask general questions after local LinkScape tools do not match.

How Linker Decides

What you ask What Linker does Example
Browser data question Uses local LinkScape SQLite data through MCP/browser tools. show my collections
Browser action Uses the running LinkScape window when available. switch to google tab
General chat question Uses your selected provider only if you saved a provider key. what is the weather today?
No local tool and no provider key Returns a standard explanation and asks you to add a key for broader chat. explain quantum computing

Tool Capabilities

Area What Linker can do
History Summarize recent activity, most visited sites, active sites today, name searches, month/year queries, and imported browser history.
Favorites List favorites, summarize counts, find saved favorite links, and open extracted links in LinkScape.
Tabs Show saved tabs, list active tabs, find a tab by title or URL, switch to a tab, open a new tab, reload, go back, go forward, and go home.
Collections List collections, show a named collection, add the current page or a URL, remove an item, and set a startup collection.
Settings Read startup behavior, home page, provider status, collection startup settings, and open-in-new-tab preferences.
AI provider keys Store user-provided keys in Windows Credential Manager. LinkScape keeps non-secret provider settings in its settings database.

Privacy Notes

  • Local browser tools read LinkScape data from local SQLite databases.
  • History, favorites, tabs, collections, and settings stay on your device unless you ask a connected provider a broader chat question.
  • Provider API keys are saved in Windows Credential Manager.
  • When a provider key is connected, local LinkScape tools still run first.
  • Safety checks can stop harmful or inappropriate chat requests before provider fallback.

Tip Type mcp status to see which local tools are available.
